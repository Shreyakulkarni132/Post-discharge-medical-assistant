# After Doctor 

The only medical consultation place where you go after your discharge. 
Helps patients get personalized attention after discharge. Built using RAG and Multi Agent System. Includes SERP API + SQLite for backend.

Project Overview
A multi-agent AI system designed to assist nephrology patients after discharge by providing personalized information retrieval and clinical support. The system uses a receptionist agent for patient interaction and data retrieval and a clinical agent for medical queries backed by RAG and web search capabilities.

Features
Multi-agent architecture with specialized roles
Patient discharge report retrieval from database
RAG-based clinical information system using nephrology reference materials
Web search integration for queries outside reference materials
Comprehensive logging system for all interactions
Source citation for all clinical information
Patient data management with error handling

How to run
run the repo by command "python app.py" from main directory after installing the requirements.txt
